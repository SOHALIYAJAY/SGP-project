# BizAnalytics Frontend

## How to run this frontend

1. **Open a terminal** and go into this project folder (don't run from the SGP root):
   ```sh
   cd insight-engine-a03a1777-Frontend
   ```

2. **Install dependencies** (once):
   ```sh
   npm install
   ```
   Or with Bun: `bun install`

3. **Start the development server**:
   ```sh
   npm run dev
   ```
   Or with Bun: `bun run dev`
